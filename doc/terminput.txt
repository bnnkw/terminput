*terminput.txt*			Input buffer for a terminal window

1. Commands		|terminput-commands|
2. Options		|terminput-options|
3. Actions		|terminput-actions|
4. Variables		|terminput-variables|
5. Mappings		|terminput-mappings|
6. Examples		|terminput-examples|

==============================================================================
1. Commands					*terminput-commands*

						*:OpenTermInput*
:OpenTermInput		Open a window where writing contents will be sent
			to the terminal.  Does nothing if the buffer
			where the command was executed is not a |terminal|.


==============================================================================
2. Options					*terminput-options*

g:terminput_mappings	Define pairs of a command name and a |keycode|.
			When the command is running in the foreground,
			|:OpenTermInput| is executed.  Otherwise, the key
			is sent to the terminal.

g:terminput_after_send	Set the action after sending contents to the
			terminal.  Defaults to clear.  See |terminput-actions|.

g:terminput_send_empty  Action to take when sending an empty buffer.
			Valid values: none, space, tab, lf, crlf.
			Defaults to crlf.
			none:  Do nothing (after-send action is not executed).
			space: Send a space (0x20).
			tab:   Send a tab (0x09).
			lf:    Send a line feed (0x0a).
			crlf:  Send a carriage return (0x0d).


==============================================================================
3. Actions					*terminput-actions*

after-send				*terminput-actions-after-send*

wipeout			Execute |:wipeout|.
clear			Execute |deletebufline()|.


==============================================================================
4. Variables					*terminput-variables*

b:terminput_term_bufnr	Point to the terminal that the buffer sends
			contents to.  Automatically set by executing
			|:OpenTermInput|.

==============================================================================
5. Mappings					*terminput-mappings*

						*terminput-CR*
<CR>			Send the contents to the terminal identified by
			|b:terminput_term_bufnr|.  The original contents
			are saved into register t.

==============================================================================
6. Examples					*terminput-examples*

>
	let g:terminput_mappings = #{
	      \ bash: '<C-O>',
	      \ fish: '<C-O>',
	      \ zsh: '<C-O>',
	      \ aider: '<C-J>',
	      \ claude: '<C-J>',
	      \ gemini-cli: '<C-J>'
	      \}
<
